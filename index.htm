Dashboard

This page allows all project team members to view data stored in an Airtable database maintained by Franklin Associates. This includes project stakeholders, elected officials, BREC staff, consultant team members, and members of the public who responded to BREC's online poll last spring.  The data also includes calendar dates of project milestones, recommended communications messages and anticipated launch dates.  This web page is temporary, intended only for display of the resources below, and will be purged when a secure - password protected - project web portal is launched.
